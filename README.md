# Curency-game-app
## General
Simple game with UI (Thymeleaf) to guess random currency exchange ratio downloaded from [remote API.](http://exchangeratesapi.io/)
## Technologies
* Java 11
* Maven
* Spring Boot 2
* Thymeleaf
* HTML, CSS
